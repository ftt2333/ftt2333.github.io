只能说这题太妙了，场上 tourist 都没过。

题目描述

  • 给定一个长度为 nnn 的序列 a1,a2…,ana_1,a_2\dots,a_na1​,a2​…,an​。
  • 对于 1≤i≤j≤n1\le i\le j\le n1≤i≤j≤n，你可以花费 (i−j)2⋅min⁡(ai,ai+1…aj)(i-j)^2\cdot\operatorname{min}(a_i,a_{i+1}\dots a_j)(i−j)2⋅min(ai​,ai+1​…aj​) 的代价从 iii 跳到 jjj。
  • 你现在要对于所有的 1≤k≤n1\le k\le n1≤k≤n，输出从 111 到 kkk 的最段路。
  • 1≤ai≤n≤4×1051\le a_i\le n\le4\times10^51≤ai​≤n≤4×105。

解题思路

首先考虑 O(n2)O(n^2)O(n2) 的 DP，fif_ifi​ 表示从 111 到 iii 的最短路径，转移很简单。

现在要考虑把这个 DP 优化到可以接受的时间复杂度内，考虑有以下性质。

性质1

对于 1≤i≤j≤n1\le i\le j\le n1≤i≤j≤n，如果直接从 iii 跳到 jjj，那么一定不会比经过区间最小值 aka_kak​ 跳两次更优。

证明很简单，如果直接从 iii 跳到 jjj，那么代价就是 ak⋅(i−j)2a_k\cdot(i-j)^2ak​⋅(i−j)2。

然而经过 kkk 的代价是 ak⋅(k−i)2+ak⋅(j−k)2=ak⋅((k−i)2+(j−k)2)a_k\cdot(k-i)^2+a_k\cdot(j-k)^2=a_k\cdot((k-i)^2+(j-k)^2)ak​⋅(k−i)2+ak​⋅(j−k)2=ak​⋅((k−i)2+(j−k)2)。

显然因为对于 a,b≥0a,b\ge0a,b≥0，有 (a+b)2≥a2+b2(a+b)^2\ge a^2+b^2(a+b)2≥a2+b2，所以经过区间最小值更优。

由此可知，每一步的区间最小值一定在两端。

性质2

对于一个最小值 aia_iai​，一定满足他前面和后面的步长一定不会超过 n/ain/a_in/ai​。

证明：

对于一个步长 ddd，如果每次只跳 111 的距离，那么每次的花费都不会超过 nnn，总花费不会超过 n⋅dn\cdot dn⋅d。

然而一步跳到的花费是 ai⋅d2a_i\cdot d^2ai​⋅d2。

要满足一步跳到更优，就有 ai⋅d2≤n⋅da_i\cdot d^2\le n\cdot dai​⋅d2≤n⋅d，既 d≤naid\le\frac{n}{a_i}d≤ai​n​。

时间复杂度

有了以上性质我们就只要对于每个 iii，考虑以 aia_iai​ 为最小值转移既可，可以证明这样的时间复杂度是 O(nn)O(n\sqrt n)O(nn​) 的。

对于 ai≥na_i\ge\sqrt nai​≥n​，步长上限 d=nai≤nd=\frac n {a_i}\le\sqrt nd=ai​n​≤n​，所以时间复杂度不会超过 O(nn)O(n\sqrt n)O(nn​)。

而对于 ai<na_i<\sqrt nai​<n​，考虑将将同的 aia_iai​ 放在一起考虑。

那么对于一个相同的 aia_iai​，以他们为最小值的区间长度之和是 O(n)O(n)O(n)。

然而最多有 n\sqrt nn​ 种数，所以时间复杂度也不会超过 O(nn)O(n\sqrt n)O(nn​)。

示例代码

代码非常好写。

#include <bits/stdc++.h>
using namespace std;
#ifndef LOCAL
void debug(...) {}
#endif
#define rep(i, n) for (int i = 0; i < (n); ++i)
#define per(i, n) for (int i = (n) - 1; ~i; --i)
#define all(v) (v).begin(), (v).end()
using ll = long long;
using vi = vector<int>;
 
int main() {
#ifndef LOCAL
  cin.tie(nullptr)->sync_with_stdio(false);
#endif
  constexpr ll inf = 1e18;
  int n;
  cin >> n;
  vi a(n);
  vector<ll> f(n, inf);
  rep(i ,n) cin >> a[i];
  f[0] = 0;
  rep(i, n) {
    int d = n / a[i];
    for (int j = i - 1; j >= 0 && i - j <= d; --j) {
      f[i] = min(f[i], f[j] + 1ll * (i - j) * (i - j) * a[i]);
      if (a[j] <= a[i]) break;
    }
    for (int j = i + 1; j < n && j - i <= d; ++j) {
      f[j] = min(f[j], f[i] + 1ll * (i - j) * (i - j) * a[i]);
      if (a[j] <= a[i]) break;
    }
  }
  rep(i, n) cout << f[i] << ' ';
  cout << '\n';
}
